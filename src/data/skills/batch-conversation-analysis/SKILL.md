# 批量对话分析 Skill (Batch Conversation Analysis)

> 大批量分析经纪人与客户的对话记录，提取总结出经纪人的难题、困惑、待提升项

**Skill 版本**：v0.1 (初始框架)
**创建时间**：2026-03-31
**来源**：加号哥需求 - 批量对话分析场景

---

## 📋 Skill 概述

这是一个**批量对话分析工具**，用于从大批量的经纪人与客户对话记录中，提取和总结共性问题。

**使用方式**：
1. 你把**大批量的经纪人与客户对话记录**发给我（可以是多个文件、一个文件夹、或者一段文本）
2. 我运用分析框架帮你**批量分析**
3. 我提取出**难题、困惑、待提升项**，并**归类**
4. 我给出**说明依据**（用具体的对话片段支撑）
5. 持续优化培训和管理方向

**核心目标**：
- 从大批量对话中发现共性问题
- 归类整理，便于针对性培训
- 用具体对话片段作为依据，说服力强
- 为团队管理和培训提供数据支撑

## 📝 核心概念

### 1. 分析维度

| 维度 | 说明 | 提取重点 |
|------|------|---------|
| **难题** | 经纪人在沟通中遇到的困难和挑战 | 卡壳的地方、搞不定的问题 |
| **困惑** | 经纪人不确定、拿不准的地方 | 犹豫、疑问、不确定的表述 |
| **待提升项** | 经纪人能力上可以提升的方向 | 沟通技巧、专业知识、流程把握 |

### 2. 归类框架

| 类别 | 说明 | 例子 |
|------|------|------|
| **需求挖掘** | 客户需求探查不深入、不全面 | 没有问清楚购房动机 |
| **产品介绍** | 省心租产品介绍不清楚、不吸引人 | 只讲功能，不讲价值 |
| **异议处理** | 客户疑问和异议回应不到位 | 被问住了，不知道怎么回答 |
| **价格谈判** | 价格相关问题处理不好 | 不敢谈价、不会议价 |
| **邀约技巧** | 面访邀约成功率低 | 客户总说"再想想" |
| **专业知识** | 市场、产品、政策等知识不足 | 回答不准确、不专业 |
| **沟通节奏** | 沟通节奏把握不好 | 太急、太慢、被客户牵着走 |
| **信任建立** | 与客户建立信任的能力不足 | 客户戒备心强、不敞开心扉 |

## 🔧 分析框架

### 阶段一：批量阅读
1. 快速浏览所有对话记录，了解整体情况
2. 识别对话类型（400电话、面访、微信等）
3. 确定时间范围和样本量

### 阶段二：标记问题
1. 逐篇阅读对话，标记问题点
2. 用不同符号标记不同类型的问题：
   - 🔴 难题（卡壳、搞不定）
   - 🟡 困惑（犹豫、不确定）
   - 🔵 待提升（技巧、知识）
3. 提取具体的对话片段作为依据

### 阶段三：归类整理
1. 把标记的问题按照归类框架分类
2. 统计每个类别的频次
3. 识别高频问题和共性问题

### 阶段四：深度分析
1. 分析每个类别的具体表现
2. 找出问题背后的原因
3. 提炼改进方向和建议

### 阶段五：生成报告
1. 整理成结构化的分析报告
2. 每个问题都附上具体的对话依据
3. 给出针对性的培训和管理建议

## 💡 使用 Tips

1. **样本量要足够**：建议至少 10-20 篇对话，才能发现共性问题
2. **关注高频问题**：频次高的问题优先解决
3. **用对话片段说话**：每个结论都要有具体的对话依据
4. **区分个人问题 vs 团队问题**：有些是个人能力问题，有些是团队共性问题
5. **持续跟踪**：定期分析，看问题是否改善
6. **结合培训**：分析结果要落地到培训和管理中
