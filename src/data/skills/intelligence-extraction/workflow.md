# 情报提取详细工作流程

> 从原始记录到结构化情报的5步工作流程

---

## 🔄 工作流程（5步）

### 第1步：阅读原始记录
- 仔细阅读客户画像和对话记录
- 标注关键信息点
- 识别客户的核心需求和顾虑

### 第2步：提取客户情报
按照以下结构提取：
1. **基本信息**：姓名、性别、年龄、性格
2. **核心需求**：需求动机、家庭成员、工作生活、房型需求、优先级、时间计划、预算能力
3. **关键顾虑**：客户的担忧和犹豫点
4. **沟通记录关键点**：重要的对话内容和决策点
5. **关键洞察**：从记录中提炼的规律和特点
6. **跟进建议**：后续跟进的方向和重点

### 第3步：提取房源情报（如果有）
按照以下结构提取：
1. **基本信息**：小区名、户型、面积、楼层、楼龄
2. **核心情报**：真实卖点、真实问题、业主底价
3. **带看记录**：客户反馈、关注点、异议
4. **竞品情报**：周边房源、价格对比
5. **业主情报**：业主心态、谈判底线

### 第4步：保存到 intelligence/ 文件夹
- 客户情报保存到：`intelligence/customer/客户姓名-YYYYMMDD.md`
- 房源情报保存到：`intelligence/listing/小区名-户型-YYYYMMDD.md`

### 第5步：复盘和总结
- 检查是否有遗漏的重要信息
- 总结从这个案例中学到的经验
- 更新 lessons.md（如果有）

## 💡 情报提取技巧

1. **抓重点**：不要逐字记录，而是提取关键信息
2. **找规律**：从对话中发现客户的行为模式和决策逻辑
3. **看细节**：注意客户的用词、语气、反应
4. **理逻辑**：把零散的信息串联起来，形成完整的客户画像
5. **提建议**：基于情报给出具体的跟进建议，不要泛泛而谈

---

## 📝 注意事项

1. **客观真实**：只记录客观事实，不添加主观猜测
2. **及时记录**：当天的情报当天记录，避免遗忘
3. **结构清晰**：按照统一模板记录，便于后续分析
4. **隐私保护**：所有情报只属于加号哥，严格保密
